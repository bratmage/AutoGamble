## AutoGamble for Turtle WoW

Requires you to actually be liked the the Bloodsails. Pirates, ahoy.

A very simple addon that automatically rolls 1g when you click on Gold-Tooth Gary.

There is a "Streak" system built into him that actually puts the earnings pretty strongly in house's favor. This addon bypasses that by auto rolling 50s when a 2x 1g streak is won in order to automatically break the streak, which keeps odds at 50:50.

Please don't kill my fun and patch this.
